How to Read the Graphs
Understanding the Graphs
SHAP Summary Plot
  • X-axis: Impact on prediction (right = more CKD risk).
  • Color: Red = high value, Blue = low value.
  • Dots = user cases; farther right = higher risk.
LIME Plot
  • Explains why this prediction was made.
  • Green bars: Push toward CKD.
  • Red bars: Push away from CKD.
PDP Plot
  • X-axis: Feature value range.
  • Y-axis: Model's predicted risk of CKD.
  • Shows how increasing a feature (e.g., creatinine) affects CKD probability.
